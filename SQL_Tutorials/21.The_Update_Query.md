# The_Update_Query

## What is it used for ?

* To update rows in a table
* You generally use the UPDATE query with the WHERE clause

## Syntax

```mysql
UPDATE table_name
SET col=val1,col2=val2,...,coln=valn
[WHERE condition];
```
